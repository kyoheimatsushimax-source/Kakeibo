# かんたん家計簿

シンプルで一目で分かる、ブラウザだけで動く家計簿アプリ。タブで「ダッシュボード / フロー / ストック」を切り替えます。

## 特徴

- **インストール不要**: `index.html` をブラウザで開くだけ
- **ダッシュボード**: 収入・支出・収支・貯蓄率のKPI (前月比つき)、収入/支出のドーナツ、カテゴリ別月別推移、年間推移、年末着地予測
- **フロー**: クレカ/固定費/変動費の3区分。固定費・変動費は項目別入力 (ドラッグ並べ替え可、固定費は翌月へ自動引継)
- **ストック**: 預金/投資/負債の口座を「前月末残高 / 今月の増減 / 今月末残高」で管理。通貨 (JPY/USD/SGD) + 月次為替で円換算。資産配分・純資産推移チャート、口座間の振替記録
- **クレカ CSV インポート**: 利用者をカード番号で自動判定、店舗名から分類を自動判別 (手動修正・ルール記憶も可)、返金(マイナス)対応、大型支出の警告
- **明細**: 検索・ソート・編集・削除 (削除はUndo可)
- **データはローカル保存**: `localStorage` に保存、外部送信なし
- **完全バックアップ**: 明細・フロー・ストック・分類ルール・表示設定をまとめてJSONでエクスポート/インポート

## 使い方

1. `index.html` をブラウザで開く
2. 上部タブで画面を切替、`‹ ›` または キーボードの `←` `→` で月を移動
3. ダッシュボードの「記録を追加」で収入/支出を入力、クレカは CSV を取込

## カテゴリ

- **収入**: 給与(K)・給与(C)・賞与(K)・賞与(C)・補助金(保育料/児童手当/区手当)
- **支出/クレカ分類**: 食費・外食・日用品・交通・娯楽・旅行・光熱費・子ども関連・医療・通信・美容・K小遣い・C小遣い・C立替・K立替・その他

## 負担と利用者

- **負担**: 家族 / K / C (デフォルト家族)。家族のみ全体集計に含む。K・Cは個人別として別集計
- **利用者**: K / C (クレカのカード番号末尾で自動判定)

## クレカ明細インポート

- 利用明細 CSV (.csv) をドラッグ&ドロップ。文字コードは UTF-8 / Shift_JIS 自動判定
- 日付・金額・店舗を自動検出、カード番号で利用者(K/C)、店舗名で分類を自動判別
- マイナス金額(返金)は相殺、平均の3倍以上の大型支出は⚠で強調
- 重複 (日付+金額+店舗+負担+利用者) は自動スキップ
- プレビューで利用者・分類を確認/修正してから一括登録
